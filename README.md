# Nova-One

Site vitrine UX/UI design, inspiré d'Apple.

## 🎨 À propos

Nova-One est un projet de site vitrine moderne qui s'inspire de l'approche de design épurée et élégante d'Apple. Ce projet met l'accent sur l'expérience utilisateur (UX/UI) et la qualité visuelle.

## 🛠️ Stack Technologique

- **TypeScript** (95.7%) - Logique applicative
- **CSS** (3.7%) - Styles et mise en page
- **JavaScript** (0.6%) - Scripts additionnels

## ✨ Caractéristiques

- Design moderne et épuré
- Interface utilisateur élégante
- Responsive et optimisé pour tous les appareils
- Code TypeScript fortement typé pour une meilleure maintenabilité
- Styles CSS personnalisés

## 🚀 Démarrage

### Prérequis

- Node.js (version 14 ou supérieure)
- npm ou yarn

### Installation

```bash
# Cloner le repository
git clone https://github.com/unrealazerty/Nova-One.git
cd Nova-One

# Installer les dépendances
npm install
# ou
yarn install
```

### Développement

```bash
# Lancer le serveur de développement
npm run dev
# ou
yarn dev
```

### Build de production

```bash
# Créer une version optimisée pour la production
npm run build
# ou
yarn build
```

Ce projet vise à démontrer les bonnes pratiques en matière de design web moderne, en combinant une esthétique minimaliste avec une expérience utilisateur intuitive, tout en maintenant un code de haute qualité.
